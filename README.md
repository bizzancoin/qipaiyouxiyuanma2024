#### 😎 软件架构

七星棋牌源码包含200多款地方游戏，UI风格比较漂亮全动态且可切换风格。

模式分为：房卡俱乐部模式和金币模式两种，任务活动齐全。

客户端采用cococs2d-js开发可以编译生成androidAPP/iosAPP/H5客户端多平台互通，并且带有小程序版本可以发布微信小程序版。

服务端采用nodejs开发，后台使用vue-admin-template框架开发，数据库mysql缓存mongodb。

全部为源代码适合大团队二次开发。
